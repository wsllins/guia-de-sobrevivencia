# Questionário de fixação
## <u>1. Por que o Aluno B teve seu comando git push recusado na fase 3?</u>
### Isso ocorreu devido a um conflito gerado por alterações feitas na mesma parte do código. Por isso, foi necessário resolver o conflito antes de dar o push corretamente.
## <u>2. Qual a utilidade real de utilizarmos comandos como git checkout -b no dia a dia de uma equipe de desenvolvimento?</u>
### A utilidade se baseia em dividir o trabalho em partes, tornando-o mais dinâmico, ágil e independente. Evitando que alterações diretas na branch principal contenha falhas e resolvendo conflitos antes de finalizar mudanças no projeto.
## <u>3. Como o Git ajuda a garantir que o código de um desenvolvedor não apague acidentalmente o trabalho de outro?</u>
### A ferramenta de versionamento auxilia nisso ao documentar conflitos e facilitar visualmente a resolução de problemas antes de permitir alterações.
